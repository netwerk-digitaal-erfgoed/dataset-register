## [0.2.0](https://github.com/netwerk-digitaal-erfgoed/dataset-register/compare/core-0.1.0...core-0.2.0) (2026-07-23)


### ⚠ BREAKING CHANGES

* **crawler:** CRAWLER_SCHEDULE is removed. The crawler now runs a single pass and exits; schedule recurring crawls externally (a k8s CronJob or cron).
* **core:** ProbeSeverities gains a required rateLimited member. Code that constructs a ProbeSeverities literal must add it; readProbeSeverities and the stage default already do.
